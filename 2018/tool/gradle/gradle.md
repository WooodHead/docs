# gradle installation

    ```
    $ mkdir /opt/gradle
    $ unzip -d /opt/gradle gradle-4.10.1-bin.zip
    $ ls /opt/gradle/gradle-4.10.1
    
    echo 'export GRADLE_HOME=/opt/gradle/gradle-2.14' | sudo tee -a /etc/profile
    ```
